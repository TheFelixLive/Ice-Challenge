## About
This challenge will help you master the ice. Every time you sneak, a layer of ice spawns beneath you.

# Installation
> [!NOTE]
> Make sure that both the Ice Challenge and the Timer are using the same CCS (Ice Challenge Communication System) Version!

1. **Download** the latest Ice Challenge release from [here](https://github.com/TheFelixLive/Ice-Challenge/releases/latest).
2. **Download** the best timer release, mostlikely from [here](https://github.com/TheFelixLive/Timer-V/releases/latest).
3. **Open** both `.mcpack` or `.mcaddon` files with Minecraft.
4. **Create** a new world OR **edit** an existing world
5. **Navigate to**: Behavior packs.
6. **Click on** "Available"
7. **Activate** the Ice Challenge & the Timer
8. `Play` or `Create` your World
